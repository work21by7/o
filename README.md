# Titanium Clock: Infinity Edition

An enterprise-grade, high-performance flip clock and productivity timer featuring Picture-in-Picture mode and focus tools.

## Features

- **Flip Clock**: Classic 3D flip animation for time, stopwatch, and timer.
- **Modes**: Clock, Stopwatch, Timer, Pomodoro (Focus), Session History, and World Clock.
- **Themes**: Multiple professional themes including Obsidian, Polar, Cyberpunk, and Forest.
- **Productivity**: Integrated session ledger with CSV export and world clock timezone interpolation.
- **Advanced Tools**:
  - Voice Commands (Wake word: "Kal")
  - Picture-in-Picture mode with cross-browser fallback.
  - Performance HUD (DevTools).
  - OLED Protection (Pixel shifting and dimming).
- **PWA**: Installable as a progressive web app for offline use.

## How to Run

To view the application locally, you can use any static file server.

### Using Python

```bash
python3 -m http.server 3000
```

Then open `http://localhost:3000` in your browser.
